==================(LEIA-ME)===================

  Esse bat automatiza as instalações para
notebook de técnicos.

  Nele poderão ser realizados as seguintes
configurações:

- Desabilitar controle de contas do user (UAC)

- Instalar os programas básicos de utilização:
	* Leitor de PDF: Adobereader
	* Navegador: Google Chrome
	* Conexao remota: Anydesk
	* Descompactador: 7zip

- Instalar os softwares de técnicos:
	* STILLCanConsole
	* LINDECanConsole
	* ZpCanFlasher
	* ZpWinConsole
	* JLW
	* Nexxt

Última atualização do batch: 10/10/2024.
